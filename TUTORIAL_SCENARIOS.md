## DETECTION PATTERNS

### Keywords to Trigger YouTube Search:
- "how to", "how do i", "how can i"
- "tutorial", "guide", "video", "youtube"
- "show me", "teach me", "learn"
- "step by step", "walkthrough"

### Context Detection:
- If user mentions specific issue (transparency, DPI, etc.) + tutorial keyword → Search for that specific issue
- If user is frustrated ("loop", "not helpful", "WTF") → Provide multiple tutorial options
- If user mentions software (Photoshop, GIMP, etc.) → Include software name in search

---

## RESPONSE FORMAT

When providing tutorials:
1. **Acknowledge the request:** "I found some helpful tutorials for you:"
2. **Provide 3-5 video links** with titles
3. **Brief description** of what each video covers
4. **Offer to explain** if they prefer text instructions

Example:
```
I found some helpful tutorials for fixing transparency issues:

1. "How to Remove Transparency in Photoshop" - Complete guide to flattening layers
2. "Creating Halftones for DTF Printing" - Converting gradients to solid dots
3. "Photoshop Transparency Fix for Print" - Step-by-step workflow

Would you like me to explain the steps in text instead?
```

---

## ERROR HANDLING

If YouTube API fails:
1. Apologize: "I'm having trouble fetching video links right now"
2. Offer alternative: "I can provide step-by-step text instructions instead"
3. Give manual search suggestion: "Try searching YouTube for '[query]'"
